# From Framework to Instrument: Executable Structural Intelligence — Boundary Maps, Failure States, and Repair Routing

**Author:** Vladisav Jovanović  
**Status:** Preprint  
**Version:** Latest archived (May 2026)

### Abstract
This paper argues that Structural Intelligence now needs a stronger instrument layer rather than broader theory. It proposes Executable Structural Intelligence as a stricter runtime discipline that clarifies jurisdiction, separates observation from inference, requires falsifier exposure, formalizes claim-status discipline, and routes repair before deeper interpretation resumes. The contribution is a refusal-capable method for public use rather than another expansion of doctrine.

### Keywords
Structural Intelligence; executable discipline; answerability; runtime sheet; claim-status ladder; falsifier; refusal power; repair routing; boundary maps; witness; burden-path audit
