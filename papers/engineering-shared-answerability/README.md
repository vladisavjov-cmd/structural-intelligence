# Engineering Shared Answerability: Design Primitives for Answerable Control

**Author:** Vladisav Jovanović  
**Status:** Preprint  
**Version:** Latest archived (May 2026)

### Abstract
This paper argues that high-performing automated systems are not yet answerable systems. Modern infrastructures can remain efficient, compliant, and visibly supervised while still becoming structurally unsafe when contradiction fails to become correction before harm is exported. The paper translates Structural Intelligence into an engineering architecture for answerable control. It proposes a layered control stack and a compact set of design primitives, including causal anchoring, adversarial mirroring, proof of contact receipt, binding revision triggers, bounded delegation, consequence-bearing authorization, burden-path audit, verifiable execution, and refusal structures. The point is not to add another ethics slogan, but to show how machine execution, human witness, and reality-contact can be bound tightly enough that correction still happens before the system protects itself by exporting burden elsewhere.

### Keywords
Structural Intelligence; shared answerability; answerable control; frictional design; bounded delegation; witness; burden-path audit; corrigibility; cybernetic systems; AI governance
