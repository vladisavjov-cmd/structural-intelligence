# The Predictive SI Runtime: Dashboard Variables, Output States, and the Falsification Ledger

**Author:** Vladisav Jovanović  
**Status:** Preprint  
**Version:** Latest archived (May 2026)

### Abstract
This paper develops the runtime layer for Predictive Structural Intelligence. It argues that breach-hazard claims become dependable only when routed through an inspectable runtime that declares domain placement, observations, time horizon, state classification, dashboard variables, hidden-holder analysis, falsifier, revision trigger, output state, downgrade rules, and ledger obligations. The paper’s central contribution is methodological: prediction becomes answerable only when the forecast itself can be corrected.
