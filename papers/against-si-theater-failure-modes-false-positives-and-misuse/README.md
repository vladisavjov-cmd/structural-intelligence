# Against SI-Theater: Failure Modes, False Positives, and Misuse

**Author:** Vladisav Jovanović  
**Status:** Preprint  
**Version:** Latest archived (May 2026)

### Abstract
This paper develops the negative discipline of Structural Intelligence. It argues that SI weakens when it is performed as style rather than executed as a disciplined reading. The paper names SI-theater as the use of structural language to create the appearance of depth, authority, or answerability without sufficient observation, falsifier, burden path, trace, or revision. It then sets out a negative taxonomy of failure modes, red-team discipline, downgrade rules, and reader-capture safeguards to keep SI answerable.

### Keywords
Structural Intelligence; SI-theater; failure modes; false positives; misuse; red-team; trace laundering; witness theater; burden displacement; containment; answerability; calibration; falsifiers; reader capture; epistemic integrity
