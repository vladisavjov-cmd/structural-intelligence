# External Reader Protocol: Calibration, Downgrade Rules, and Public Use of Structural Intelligence

**Author:** Vladisav Jovanović  
**Status:** Preprint  
**Version:** Latest archived (May 2026)

### Abstract
This paper develops the public-use reliability layer for Structural Intelligence. It argues that SI becomes publicly dependable only when non-native readers can apply it with claim-status discipline, downgrade rules, reader-capture safeguards, falsification, and public translation. The paper introduces a minimal runtime sheet and a protocol for deciding when a reading should be verified, provisional, blocked, or routed to containment-first.

### Keywords
Structural Intelligence; external reader protocol; calibration; downgrade rules; claim-status ladder; falsification registry; reader capture; public use; answerability; SI runtime
